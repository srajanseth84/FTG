## Run Locally
------

* Clone the project

```bash
  git clone https://github.com/srajanseth84/FTG.git
```

* Go to the project directory

```bash
  cd FTG
```
* Create venv

```bash
  python3 -m virtualenv venv 
```

* Activate the venv

```bash
  source venv/bin/activate
```

* Install dependencies

```bash
  pip install -r requirements.txt
```

* Start the server

```bash
  streamlit run app.py 
```
